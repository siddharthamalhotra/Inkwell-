Add a webhook to your GitHub repo so the wiki regenerates automatically whenever a significant commit is pushed.

  1. Go to your repo → Settings → Webhooks → Add webhook
  2. Paste the URL above as the Payload URL
  3. Set Content type to application/json
  4. Under "Which events", choose Just the push event
  5. Optionally set a Secret and add WEBHOOK_SECRET=<same value> to your .env
